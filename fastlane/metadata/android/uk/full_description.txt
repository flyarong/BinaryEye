Працює в книжковій та альбомній орієнтації, може читати перевернуті коди,
інтерфейс Material Design, а також може генерувати штрих-коди.

Використовує бібліотеку сканування штрих-коду ZXing ("Zebra Crossing").
Підтримувані формати штрих-коду: AZTEC, CODABAR, CODE 39, CODE 93, CODE 128,
DATA MATRIX, EAN 8, EAN 13, ITF, PDF417, QR CODE, RSS 14, RSS EXPANDED,
UPC A, UPC E and UPC EAN EXTENSION.

Код додатку: https://github.com/markusfisch/BinaryEye
